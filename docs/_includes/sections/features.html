Application Features

Trax brings all your issues across multiple repos onto one Kanban style board. Velocity planning, invoice generation and time reports are all baked in. Use it as an all-in-one solution, or turn off the features you don't care about.

A single unified board

Track issues across multiple github repos on a single board. Similar to github projects, but with more power and elegance.

Time tracking on issues

Use your current github issues for time tracking, invoice generation, velocity planning, and burn rate reports.

Work offline

Tech nomads tend to wander. Trax synchronizes your data locally so you can manage issues and track time offline.
