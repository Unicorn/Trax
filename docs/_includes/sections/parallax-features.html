Project management for github

Whether your a freelancer, SCRUM master, or team developer: if you use github then you will love Trax. With Trax you can properly manage multi-task your pet-projects, open source contributions, and clients!

Velocity Planning

Built-in velocity planning using a fibonacci weight scale and github issue labels.

Story Templates

Use the default issue templates to create and distinguish: epics, stories, features and bugs.

Issue Timers

Support for time tracking on any github issue in addition to generic timers.
